# UI/UX 規格文件

**設計風格：** 暗色系（Dark Mode）
**主色：** 紫色 #7C3AED
**參考：** BeatStars、Spotify 暗色版

---

## 1. 設計原則

- **沉浸感**：深色底讓音樂封面圖更突出
- **直覺操作**：試聽、購買動線一目瞭然
- **品牌感**：製作人有足夠空間展示個人風格
- **行動優先**：手機上同樣好用

---

## 2. 配色系統

```
背景層次：
  最深背景：#0F0F0F（整頁底色）
  卡片背景：#1A1A1A
  卡片懸停：#242424
  邊框：#333333

主色調：
  Primary：#7C3AED（紫）
  Primary Hover：#6D28D9
  Secondary：#10B981（綠，用於 "Now Playing"）

文字：
  主文字：#FFFFFF
  次要文字：#A0A0A0
  禁用文字：#555555

狀態色：
  成功：#10B981
  警告：#F59E0B
  錯誤：#EF4444
  Sold Out：#6B7280
```

## 3. 頁面設計規格

### 3.1 首頁（/）

```
[Navbar]
  Logo | Beats | Producers | 購物車 icon | 登入

[Hero Section]
  - 大標題：「Find Your Sound」

  - 橫向捲動卡片
```

| 裝置 | 寬度 | Beat Grid |

| 平板 | 640-1024px | 2欄 |
| 桌機 | > 1024px | 4欄 |

---

## 5. 動畫規格

| 元素 | 動畫 |
|------|------|
| 卡片 hover | 輕微上移 translateY(-4px) + shadow |
| 播放按鈕 | 封面遮罩淡入 + 按鈕縮放 |
| 頁面切換 | fade in 200ms |
| 購物車計數 | bounce 動畫 |
| 篩選側欄（手機）| slide in from left |
