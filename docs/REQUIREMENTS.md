# 需求規格書（SRS）

**專案名稱：** BeatMarketplace
**版本：** v1.0
**日期：** 2026-03-30
**狀態：** 草稿

---

## 1. 專案背景與目標

### 1.1 背景
本平台由音樂製作人團隊發起，旨在建立一個專屬於華語音樂製作人的伴奏交易與作品展示平台，補足台灣市場目前缺乏本地化 BeatStars 類型服務的空缺。

### 1.2 目標
- 提供製作人一個專業的線上形象展示空間
- 建立伴奏的線上購買與授權交易機制
- 促進製作人與詞曲創作者之間的直接合作媒合

---

## 2. 使用者角色

### 2.1 製作人（Producer）
- 可註冊帳號並建立個人資料卡
- 可上傳作品集（免費展示）
- 可上傳伴奏並設定售價及授權方案
- 可接收購買通知及客製化詢問
- 可管理自己的作品與訂單

### 2.2 買家（Buyer）
- 無需註冊即可瀏覽、試聽
- 需提供 Email 才能購買
- 可搜尋、篩選伴奏
- 可加入購物車、完成付款
- 可聯絡製作人詢問客製化服務

### 2.3 訪客（Visitor）
- 可瀏覽所有公開內容
- 可試聽所有伴奏（限制時長或完整，由製作人決定）

## 4. 非功能需求

| 項目 | 要求 |
|------|------|
| 回應速度 | 頁面載入 < 3 秒 |
| 音訊串流 | 試聽不需等待完整下載 |
| 行動裝置 | 響應式設計，支援手機瀏覽 |
| 安全性 | 密碼加密（bcrypt）、HTTPS |
| 付款安全 | 不儲存信用卡資訊，交由金流處理 |
| 瀏覽器相容 | Chrome、Safari、Firefox 最新版 |

---

## 5. 授權方案說明

| 授權類型 | 說明 | 常見限制 |
|----------|------|---------|
| Basic License | 最基本授權，用於非商業或小規模發行 | 發行量 < 2,000、不可用於 YouTube 商業化 |
| Premium License | 中階商業授權 | 發行量 < 10,000、可 YouTube 商業化 |
| Exclusive License | 獨家授權，買家取得完整所有權 | 無限制，賣出後其他人無法購買 |

---

## 6. 限制與假設

- 初期只支援台灣製作人，UI 以繁體中文為主
- 音樂檔案儲存於本地伺服器（D 槽），未來可遷移至雲端
- 綠界金流需要商業登記，開發階段使用測試模式
- 不含版稅分潤系統（未來版本規劃）
